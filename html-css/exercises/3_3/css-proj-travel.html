Viagem para Villa La Angostura

Villa La Angostura é uma cidadezinha próximo a Bariloche, aproximadamente 1h de ônibus ou 2 horas de Catamarã. Nessa incrível cidadezinha, podemos ver como a Patagônia Argentina é bela!
Ao chegarmos no porto, onde o Catamarã desembarca já podemos ver:

  • As águas cristalinas do lagos (conseguimos ver aproximadamente 3m de profundidade!!)
  • As montanhas cobertas de neve(caso tenha ido no inver)
  • Etc,etc,etc
